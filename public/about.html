Cara Kerja

Tiga langkah menuju review CV yang sesungguhnya

Seluruh proses berlangsung dalam hitungan menit, mulai dari upload file hingga laporan review yang siap Anda tindaklanjuti.

01
📂

Isi konteks lamaran & upload CV

Masukkan target posisi, bidang industri, lama pengalaman, dan tempel job description dari lowongan yang ingin Anda lamar. Upload file CV dalam format PDF atau DOCX maksimal 5MB.

→
02
🔍

Ekstraksi konten di browser Anda

File CV dibaca dan diekstrak langsung di browser menggunakan pdf.js dan mammoth.js. Tidak ada file yang diunggah ke server kami — hanya teks hasil ekstraksi yang dikirim untuk dianalisis, itupun hanya sekali dan tidak disimpan.

→
03
🧠

AI membaca CV seperti HR Professional

Teks CV dianalisis oleh model AI terbaik yang tersedia — Gemini, Groq, Mistral, atau fallback otomatis. Hasilnya bukan sekedar saran umum, melainkan review menyeluruh dengan skor, temuan kritis, risiko penolakan, dan rencana perbaikan yang konkret.

Kenapa berbeda?

Bukan sekadar "CV Checker" biasa

See My CV tidak hanya memeriksa format atau ejaan. Sistem kami mensimulasikan cara seorang HR Professional menilai CV — mempertimbangkan konteks industri, level jabatan, job description, dan sinyal-sinyal halus yang sering menjadi alasan CV ditolak tanpa penjelasan.

Privasi & Keamanan

Data Anda tidak pernah kami simpan

CV berisi data pribadi yang sensitif. Kami merancang See My CV dengan prinsip privacy by design — bukan hanya kebijakan, tapi arsitektur teknis yang memastikan data tidak bisa disimpan meski kami mau.

🔒

Ekstraksi di browser, bukan server

File CV Anda tidak pernah diunggah ke server kami. Proses membaca dan mengekstrak teks dari PDF atau DOCX terjadi sepenuhnya di peramban browser Anda menggunakan pdf.js dan mammoth.js.

⚡

Pemrosesan real-time, tanpa database

Teks hasil ekstraksi dikirim ke fungsi serverless, diproses secara real-time oleh AI, lalu hasilnya langsung dikirim kembali. Tidak ada database yang menerima atau menyimpan data CV Anda.

🚫

Zero persistence setelah respons dikirim

Setelah hasil review ditampilkan di browser Anda, data CV tidak tersisa di mana pun pada sisi server aplikasi. Setiap sesi adalah sesi baru yang berdiri sendiri.

🛡️

API key aman di sisi server

Kunci API untuk layanan AI tidak pernah terekspos ke sisi browser. Seluruh komunikasi dengan AI dilakukan melalui serverless function yang hanya menerima request dari origin yang diizinkan.

Teknologi

Dibangun di atas teknologi yang tepat

Arsitektur static + serverless tanpa framework berat — ringan, cepat, dan mudah di-maintain. Setiap komponen dipilih untuk keandalan dan keamanan.

Builder · See My CV

See My CV lahir dari frustrasi melihat alat review CV yang ada terlalu generik dan tidak memberikan gambaran nyata tentang bagaimana recruiter membaca sebuah CV. Aplikasi ini dibangun dengan filosofi sederhana: bantu pencari kerja melihat CV mereka dari sudut pandang yang benar — kritis, kontekstual, dan dapat ditindaklanjuti. Untuk pertanyaan dan masukan, kirim ke support@seemycv.app.
