Formulir Persetujuan Pemrosesan Data Pribadi

Layanan Evaluasi Curriculum Vitae (CV)

Sebelum menggunakan See My CV, Anda perlu membaca dan menyetujui ketentuan pemrosesan data pribadi berikut. Layanan ini memproses CV hanya untuk menghasilkan audit, evaluasi, dan rekomendasi perbaikan CV secara otomatis.

01 Tujuan

Analisis format, susunan, substansi, dan kesesuaian CV dengan target posisi.

02 Data

Informasi yang Anda cantumkan secara sadar dan sukarela di dalam CV.

03 Proses

Real-time, sementara, dan tidak disimpan dalam database aplikasi.

1. Tujuan Pemrosesan

Dengan menggunakan layanan ini, Saya menyatakan persetujuan untuk memberikan dokumen dan/atau data pribadi Saya yang terdapat dalam CV untuk diproses melalui See My CV. Pemrosesan dilakukan untuk tujuan analisis format, evaluasi susunan, penilaian kesesuaian dengan target posisi, dan pemberian ulasan substansi atas Curriculum Vitae yang Saya unggah.

2. Jenis Data Pribadi yang Diproses

Data pribadi yang diproses adalah informasi yang secara sadar dan sukarela Saya cantumkan di dalam dokumen CV yang diunggah. Saya memahami bahwa CV dapat memuat data seperti nama, alamat email, nomor telepon, riwayat pendidikan, pengalaman kerja, keterampilan, portofolio, dan informasi profesional lainnya.

Saya disarankan untuk tidak mencantumkan data pribadi yang bersifat spesifik atau tidak relevan dengan kepentingan rekrutmen profesional, termasuk tetapi tidak terbatas pada NIK, ras, agama, data kesehatan, informasi finansial, atau data sensitif lainnya.

3. Alur Pemrosesan Data

Saya memahami bahwa untuk menyelenggarakan evaluasi CV, sistem akan membaca dan menganalisis teks dari CV yang Saya unggah. File CV diparse di peramban browser, kemudian teks hasil ekstraksi dikirim ke fungsi serverless untuk diproses secara otomatis.

  • Data dapat diproses melalui penyedia infrastruktur aplikasi, yaitu Netlify, semata-mata untuk menjalankan layanan web dan fungsi serverless.
  • Data dapat diteruskan ke layanan AI pihak ketiga yang digunakan oleh See My CV semata-mata untuk menghasilkan ulasan CV secara otomatis.
  • Pemrosesan dilakukan secara langsung, real-time, dan sementara selama aktivitas evaluasi CV berlangsung.

4. Penyimpanan dan Penghapusan Data

Saya memahami bahwa See My CV tidak melakukan penyimpanan terhadap dokumen CV maupun data pribadi Saya dalam database atau media penyimpanan permanen aplikasi. Data hanya diproses sementara untuk menghasilkan hasil review dan tidak disimpan oleh aplikasi setelah respons ditampilkan pada browser Saya.

5. Hak Saya sebagai Subjek Data Pribadi

Karena sistem tidak menyimpan data secara permanen, penghapusan data pada sisi aplikasi terjadi secara otomatis setelah proses evaluasi selesai. Saya memahami bahwa Saya dapat menarik persetujuan dan menghentikan pemrosesan kapan saja dengan menutup halaman situs sebelum proses evaluasi selesai.

Untuk pertanyaan mengenai pemrosesan data pribadi pada layanan ini, Saya dapat menghubungi pengelola layanan melalui: support@seemycv.app.

6. Pernyataan Keabsahan dan Tanggung Jawab

Saya menyatakan bahwa seluruh data yang Saya unggah diberikan secara sadar, sukarela, dan tanpa paksaan dari pihak mana pun. Apabila Saya mencantumkan data pribadi pihak lain, termasuk tetapi tidak terbatas pada kontak referensi atau pemberi rekomendasi, Saya menjamin telah memperoleh izin dari pemilik data tersebut.

Saya bertanggung jawab penuh atas keputusan Saya untuk mengunggah dokumen tersebut. Jika terdapat data yang tidak relevan atau bersifat sensitif di dalam CV, Saya memahami bahwa hal tersebut merupakan tanggung jawab Saya sebagai pengguna untuk melakukan penyaringan sebelum mengunggah dokumen.
